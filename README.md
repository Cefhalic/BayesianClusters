# Bayesian Clusters

## To build the executable and generate the documentation
```
make
```
The executable is then at `./Cluster.exe' and the generated documentation is at `doxygen/html/index.html'.

## To see help
```
./Cluster.exe --help
```
## To run
```
./Cluster.exe --cfg config.txt -i 1_un_red.csv
```
